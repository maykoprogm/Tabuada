# Tabuada em Python

Este projeto é um programa simples em Python que gera a tabuada de um número fornecido pelo usuário. 
O programa solicita ao usuário que digite um número e exibe a tabuada desse número com 10 termos.

## Funcionalidades

- Solicita ao usuário que digite um número.
- Mostra a tabuada completa do número, de 1 a 10.
- Simples, rápido e fácil de usar.

## Como usar

1. Certifique-se de ter o Python instalado na sua máquina (recomendado Python 3.10 ou superior).
2. Clone este repositório ou baixe o arquivo `tabuada.py`.
3. No terminal ou no PyCharm, execute o programa com o comando:

```bash
python tabuada.py
